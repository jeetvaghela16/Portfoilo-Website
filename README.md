# Portfolio Website

Welcome to the personal portfolio of **Jeet Vaghela**, an IT Engineer and Software Developer.

## Features
- **Modern UI**: Clean, responsive design with a dark/light theme aesthetic.
- **Dynamic Background**: Custom particle network animation.
- **Interactive Forms**: Contact form with toast notifications and Formspree integration.
- **Project Showcase**: Grid layout for displaying projects.

## Technologies
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Design**: Custom SVG Logos, Font Awesome Icons, Google Fonts
- **Deployment**: Static hosting (GitHub Pages ready)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
